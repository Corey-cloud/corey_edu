# 在线教育系统

#### 介绍
**在线教育系统后端**
1.  技术栈：SpringBoot + SpringCloud + MyBatis-Plus + HttpClient + MySQL + Maven + EasyExcel + nginx
2.  其中涉及到的中间件包括Redis、阿里云OSS、阿里云视频点播
3.  业务中使用ECharts做图表展示、使用EasyExcel完成分类批量添加、分布式单点登录使用了JWT
